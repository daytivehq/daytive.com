{{< timelineItem icon="code" header="Browser Extension > Marklog" badge="JavaScript" >}}
<b>Marklog</b>는 기존 브라우저 즐겨찾기의 불편함을 해소하기 위해 만들어졌습니다. 새 탭을 열면 내가 등록한 즐겨찾기와 폴더, 사이트등을 한 눈에 볼 수 있습니다. </br>
큰 화면에 최적화된 대시보드 형태로 구성했으며, "/"을 눌러 초성 검색으로 빠르게 사이트에 접근 가능합니다. 키보드만으로 즐겨찾기와 사이트를 이동시켜서 마우스 없이 사이트 실행이 가능합니다.

{{< timelineItem icon="code" header="Flow Launcher Plugin > AliasFlow" badge="C#">}}
<b>AliasFlow</b>는 <a href="https://www.flowlauncher.com/" target="_blank">Flow Launcher</a> 전용 플러그인으로, 사용자가 정의한 키워드를 통해 웹사이트, 로컬 프로그램, 전역 단축키를 빠르게 실행할 수 있도록 도와줍니다. </br>
한글 초성 검색(예: ㄴㅇㅂ)과 GUI 기반 설정을 지원하여 복잡한 경로와 단축키를 기억하지 않고 등록한 키워드만으로 빠르게 검색하여 실행가능합니다. </br></br>
